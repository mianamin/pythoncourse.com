Explanation

In Python, "input" is a built-in function used to receive input
from the user via the keyboard. When you use the
"input" function, it displays a prompt (if provided) and waits for the
user to enter some text followed by pressing the Enter key.
The input provided by the user is returned as a string.

Examples:

name = input("Please enter your name: ")
address = input("Please enter your address: ")
age = int(input("Enter your age: "))
print("Your name is ",name)
print("Your address is ",address)
print("Your age is ",age)
