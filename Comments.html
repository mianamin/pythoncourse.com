Comments

Comments are used to explain Python Code.
There are two types of comment.

Single line Comment

In Python, a single-line comment is a piece of text that is ignored by the Python
nterpreter when executing the code. It is typically used to provide explanations,
notes, or annotations within the code. Single-line comments begin with the hash symbol (#)
and extend to the end of the line. Here's an example:



# this is single line comment
# this is also a single line comment
print("Hello world!")
# this is last line of program


print("Hello world!")
# this line print Hello world!



Multi Line Comment

In Python, there isn't a built-in syntax specifically for multi-line comments like there is in some
other programming languages. However, you can use triple quotes (''' or """) to create
multi-line string literals, and they are often used as a workaround for multi-line comments.
Since Python ignores string literals that are not assigned to a variable or used in any way, you
can effectively use triple-quoted strings as multi-line comments.

Here is an example:




""" this is multi-line comment
this is second line of comment
this is third line of comment
this is last line of comment
"""
print("Hello world!")
""" text between these quotes are ignored by the compiler
the purpose of comments is to explain the program"""
