Variables






Variable Name:

A variable name can only contain alpha-numeric characters and
underscores (A-z, 0-9, and _ )
A variable name must start with a letter or the underscore character
A variable name cannot start with a number
Variable names are case-sensitive ( address, Address and Address
are three different variables)




In case of multiple words, we can use either technique for better readability.

Camel Case:

myvariablename = "Mian Amin Shah"
age = 19

Pascal Case:

MyVariableName = "Mian Amin Shah"
Age = 19

Snake Case:

my_variable_name = "Mian Amin Shah"
my_age = 19

Assign one value to Multiple Variables:

a = b = c = 19

Assign many values to Multiple variables:

name , address , age = "Mian Amin Shah" , "Charsadda" , 19
